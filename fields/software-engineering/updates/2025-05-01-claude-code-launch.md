# Claude Code Launch

Anthropic released Claude Code, a terminal-first AI coding agent that plans, writes, tests, and commits code autonomously. Unlike IDE-based copilots, Claude Code operates directly in the terminal, working with any editor and codebase.

## Key Details

- **Terminal-first approach** — runs in the command line, agnostic to editor/IDE
- **Agentic workflow** — autonomously plans multi-step changes, writes code, runs tests, and creates commits
- **Powered by Claude models** — uses Anthropic's Opus and Sonnet models, which went on to dominate coding task mentions in the 2026 Pragmatic Engineer survey

## Impact

Within 8 months of launch, Claude Code became the #1 most-used coding tool among Pragmatic Engineer survey respondents (900+ engineers, March 2026). 46% of respondents named it "most loved." This rapid adoption — from zero to market leader — is one of the fastest tool category shifts in software engineering history.

Source: [Pragmatic Engineer AI Tooling Survey, Mar 2026](https://newsletter.pragmaticengineer.com/)
